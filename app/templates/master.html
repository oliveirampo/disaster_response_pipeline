Web App for Disaster Response

This web app uses machine learning to categorize messages that were sent during disaster events.

It also receives new messages and gets classification results based on 35 categories. See below the list of categories.

The categorization of disaster events is useful to provide concise and helpful information to disaster relief agencies.
